# LITPER Tracker

App flotante de escritorio para registro rapido de trabajo.

## Caracteristicas

- Ventana flotante siempre visible
- Contadores rapidos con +/- (click derecho = +5/-5)
- Timer con alertas de colores
- Modo mini y super-mini
- Persistencia local
- Atajos de teclado (Ctrl+Shift+P)

## Desarrollo

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run electron:dev

# Construir para produccion
npm run build
```

## Controles

| Accion | Metodo |
|--------|--------|
| +1 | Click en [+] |
| +5 | Click derecho en [+] |
| -1 | Click en [-] |
| -5 | Click derecho en [-] |
| Editar | Click en numero |
| Mostrar/Ocultar | Ctrl+Shift+P |
